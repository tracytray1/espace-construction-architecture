

# Espace Construction Architecture 🏛️  

Website for **Espace Construction Architecture**, built with **React + Vite** and deployed on **Netlify**.  
The project showcases architectural works, an "About" section, and a dynamic project portfolio with smooth navigation.  

---

## 🚀 Tech Stack  

- [React](https://react.dev/) — UI and components  
- [Vite](https://vitejs.dev/) — Fast build tool  
- [React Router](https://reactrouter.com/) — Page routing  
- [Framer Motion](https://www.framer.com/motion/) — Animations  
- [SCSS](https://sass-lang.com/) — Modular styling  
- [Netlify](https://www.netlify.com/) — Hosting & continuous deployment  

---

## 📦 Installation  

1. Clone the repository:  
   ```bash
   git clone https://github.com/your-username/your-repo.git
   cd your-repo
   ```
Install dependencies:

   ```bash
npm install
   ```
Start the development server:

   ```bash
npm run dev
   ```
👉 The app will be available at http://localhost:5173.

## 🛠️ Available Scripts
   ```bash
npm run dev → start local dev server

npm run build → build the app for production

npm run preview → preview the production build locally
   ```

## 📁 Project Structure
   ```bash
src/
 ├─ assets/              # Images, logos, icons
 ├─ components/          # Reusable components
 ├─ pages/               # Main pages (Home, Projects, About)
 ├─ styles/              # Global SCSS files
 ├─ projects-data.js     # Project data
 ├─ main.jsx             # React entry point
 └─ App.jsx              # Router + main layout
   ```
